Unofficial FreeBSD.org pkg mirror hosted by Yandex Cloud BareMetal

Keep (make sure) signature_type and fingerprints attributes set to correct values when using this mirror.

All architectures are mirrored. To use this mirror, one can use a custom /usr/local/etc/pkg/repos/FreeBSD.conf pkg configuration file:

FreeBSD-ports: { enabled: no }
FreeBSD-ports-kmods: { enabled: no }
FreeBSD-base: { enabled: no }

ya-FreeBSD-ports: {
  url: "http://pkg.freebsd.yaoff.net/${ABI}/quarterly",
  mirror_type: "http",
  signature_type: "fingerprints",
  fingerprints: "/usr/share/keys/pkg",
  enabled: yes
}
ya-FreeBSD-ports-kmods: {
  url: "http://pkg.freebsd.yaoff.net/${ABI}/kmods_quarterly_${VERSION_MINOR}",
  mirror_type: "http",
  signature_type: "fingerprints",
  fingerprints: "/usr/share/keys/pkg",
  enabled: yes
}
ya-FreeBSD-base: {
  url: "http://pkg.freebsd.yaoff.net/${ABI}/base_release_${VERSION_MINOR}",
  mirror_type: "http",
  signature_type: "fingerprints",
  fingerprints: "/usr/share/keys/pkgbase-${VERSION_MAJOR}",
  enabled: no
}

There are some logs per each update cycle:

  • wget repo metadata update
  • RELENG_13 update status
  • RELENG_14 update status
  • RELENG_15 update status
  • RELENG_16 update status
Each RELENG_* log file contains a list of repositories that were not updated due to an error.


Scripts used for this mirror are published GitHub. PRs are welcome.
